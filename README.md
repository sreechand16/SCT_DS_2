# 🛳️ Data Science Internship – Task 2 (Titanic Dataset EDA)

## 📝 Task Description
As part of my Data Science internship, I performed **data cleaning and exploratory data analysis (EDA)** on the Titanic dataset from Kaggle.

## 📁 Files Included
- `train.csv` – Titanic dataset  
- `Task 2.ipynb` – Jupyter notebook containing full analysis  
- `README.md` – Project documentation  

## 📊 Key Analysis
- **Survival distribution** by gender, passenger class, and age  
- **Missing value imputation** for Age & Embarked  
- **Correlation heatmap** of numerical features  
- **KDE plots** showing survival trends  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Seaborn  
- Matplotlib  
- Jupyter Notebook  

This project demonstrates how to **clean and analyze a dataset before predictive modeling**.
